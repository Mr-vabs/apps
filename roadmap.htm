100 Days Of Coding Challenge

Here's A Roadmap To Learn Coding In 100 Days or 14 Weeks

  • Beginning from HTML and CSS

    In the very beginning , we will learn HTML from any source which can be comfortable as for person. Learning HTML by your comfortable resources can be productive. Remember learning is not enough alone , you should practicing your skills by coding and this is what we do in this challenge. Learn CSS also along with HTML because practicing HTML only will not be enough and CSS can increase your motivation. For first two weeks , learn HTML and CSS and don't forgot to share progress and codes. HTML may not took more than 4 days depends upon source as well so main focus will be good CSS practice.

    Week 1st to 2nd

  • Medium level with JS and Responsive Design

    For the 3rd and 4th week , we'll learn JavaScript and Responsive Design. First two weeks we've learned HTML and CSS so JS is the next one. With HTML, CSS and JS we can also make responsive design which is a good practice and often recommended by everyone. JS is a scripting language and it has a huge industry so it need sometime with proper determination to learn. Responsive Design is combination of all three so it's better to practice it after learning JS. JS can help you to make awesome interactive websites , web apps as well as ML and AI. It also has many frameworks which will be target of next period.

    Week 3rd to 4th

  • Advance level with JS and it's Frameworks

    For the next time period week 5th and 6th , we will learn advance JavaScript and some of its popular frameworks. JS is used for many purposes like website , ( static , dynamic and responsive ) , web apps , apps , AI and so on. It has many frameworks for various purposes so its normal to be distracted. No worries as our first target is to become a front end expert then after we plan for full stack and others. So we will learn such frameworks which are used to make interactive design. Advance JS can help to make web games with AI. We have some best examples which we will share with you to encourage when it requires.

    Week 5th to 6th

  • Expert level with some dev tools

    Now we've some good knowledge of Front end development. Our focus will be on some dev tools which are used to increase productivity and expertise to make you a professional developer. Git

    Week 7th

  • Now we move to full stack development

    Full stack developers are experts who knows all about web development. They knows front end development , backend development and database administration. So it's require lot time for learning full stack technology. It can be learned with anything such as JS , MERN , Python , Php and so on. So take time to choose correct technology for learning. Since it has long time it can take upto 3 weeks and have a good dynamic project.

    Week 8th to 10th

  • After Web development , we'll move to Programming and App development

    These are also huge topics and it also requires a lot time. We prefer to give it atleast 4 weeks where we can choose one programming language to learn basic programming concepts , DSA then OOP. It can be done in 10 to 14 days which will be equivalent to 2 weeks. 2 weeks for app development is less but we've may have learned full stack development so it will not take much. Don't we will be ready for every situations and problems. We all are concerned to help each other maybe we can't do it alone but definitely we can do it together.

    Week 11th to 14th
